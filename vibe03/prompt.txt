좋은 질문입니다! 확인해보니 부드러운 스크롤은 CSS에서 처리하고 있습니다:

JS 파일을 포함했을 때 vs 안 했을 때 차이:
✅ JS 없이도 동작하는 기능 (CSS 덕분)
✅ 부드러운 스크롤 - scroll-behavior: smooth
✅ 호버 효과 - :hover 가상 클래스
✅ 애니메이션 - CSS @keyframes
✅ 링크 이동 - HTML <a href="#section">
❌ JS가 있어야 동작하는 기능
❌ 네비게이션 자동 하이라이트 - 스크롤 위치에 따라 현재 섹션 표시
❌ 스크롤 애니메이션 - 스크롤 시 요소가 페이드인되며 나타남
❌ 스크롤 진행 바 - 상단에 페이지 읽기 진행도 표시
❌ 파티클 배경 - 인트로 섹션 동적 파티클
❌ 인터랙티브 호버 - 태그/카드 클릭 시 특수 효과
❌ 모바일 메뉴 - 햄버거 메뉴 토글
현재 JS를 주석처리해도 기본 기능은 모두 동작하지만, 인터랙티브한 사용자 경험은 줄어듭니다!
