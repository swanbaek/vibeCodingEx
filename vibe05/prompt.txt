구현 내용

무한 스크롤: 페이지를 스크롤하면 자동으로 다음 포스트 6개씩 로드(총 30개까지).
로딩 상태: 새 포스트를 가져오는 동안 "로딩 중..." 표시.
종료 표시: 모든 포스트를 확인하면 "모든 게시물을 확인했습니다" 메시지.
초기 로드: 앱 시작 시 자동으로 첫 6개 표시.
작동 방식

IntersectionObserver가 피드 하단 sentinel 요소를 감지.
10% 이상 화면에 보이면 다음 페이지 로드(0.6초 딜레이로 실제 API처럼).
각 포스트는 고유 seed로 다른 아바타/이미지 생성.

-----------------------------------

구현 내용

댓글 입력: 각 포스트 하단에 댓글 입력 필드 + "게시" 버튼.
실시간 등록: 입력 후 엔터 또는 게시 버튼으로 즉시 댓글 추가.
댓글 토글: "댓글 N개 모두 보기/숨기기" 버튼으로 표시/숨김 전환.
댓글 목록: 사용자 이름(굵게) + 댓글 내용, 최대 200px 높이로 스크롤.
상태 관리: 각 포스트별로 독립적인 댓글 배열 유지.
주요 기능

빈 댓글은 게시 불가(버튼 비활성).
댓글 작성자는 'me'로 표시.
댓글 개수 실시간 업데이트.
